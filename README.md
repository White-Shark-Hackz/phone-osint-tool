## 🔍 Features
- ✔ Phone number validation
- 🌍 Country detection
- 📡 Carrier info
- 🕒 Timezone info

## 🚀 Installation (Kali Linux)

```bash
git clone https://github.com/White-Shark-Hackz/phone-osint-tool.git
cd phone-osint-tool
python3 -m venv venv
source venv/bin/activate
